This project explores how college-level attributes and draft characteristics translate into long-term NFL success, using Pro Bowl selections as the benchmark for elite performance. Built in Python using Google Colab, the project merges large-scale college and NFL datasets (~2,000 player records) to analyze which factors truly matter when evaluating draft prospects.
